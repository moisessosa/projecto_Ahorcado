As Reglas São 👮

Descrição geral. A linha tracejada, a palavra gerada aleatoriamente é representada para você adivinhar, os traços mostram o número de letras. Cada vez que você comete um erro, você perde uma oportunidade e uma seção da forca é desenhada. Você deve adivinhar a palavra antes que as oportunidades se esgotem e o desenho esteja completo.

    Observaçoes:
  • As palavras foram formatadas, retirando-se sinais de pontuação como '´', '~', '^', para facilitar a escrita.
  • Também o 'Ç' foi substituído por 'C' para compatibilidade com teclados de outras línguas
  • Você está usando uma API gratuita que gera as palavras em português, note que algumas palavras possuem erros de grafia, reconheço que é um problema adivinhar as palavras.
